# 🍎 Apple-Style Calculator

A sleek, functional web calculator inspired by the iOS design.

[Live Demo](https://alexdev262.github.io/Apple-Calculator/)

## 🚀 Features
* **PEMDAS Logic:** Correctly handles order of operations (Parentheses, Exponents, Multiplication, Division, Addition, Subtraction).
* **Keyboard Support:** Fully interactive using your physical keyboard for faster calculations.
* **Smart Backspace:** Just click the digital display to remove the last character.
* **Responsive Design:** Styled with CSS to mimic the iconic Apple aesthetic.

## 🛠️ Built With
* **HTML5** - Structured for accessibility.
* **CSS3** - Custom properties and Flexbox for that frosted-glass look.
* **JavaScript (Vanilla)** - Pure logic without external libraries.

## ⌨️ Keyboard Shortcuts
| Key | Action |
| :--- | :--- |
| `0-9` | Numbers |
| `+ - * /` | Operators |
| `Enter` | Equals (=) |
| `Backspacing / Click Display` | Delete last digit |
| `Delete` | Clear All |
